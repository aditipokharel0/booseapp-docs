Runs a BOOSE program and returns the final interpreter context.
Syntax
C#
    public InterpreterContext Run(string source, ICoreCanvas canvas, ICoreOutput output)
Parameters
source
The BOOSE program source as a multi-line string.
canvas
Canvas implementation used by drawing commands.
output
Output implementation used for program output.
Return Value
The interpreter context containing variables, arrays, and runtime settings.
Exceptions
Exception Condition
Exception Thrown when the source contains invalid syntax or missing end statements (for example: IF missing END IF, WHILE missing END WHILE, METHOD missing END METHOD).
